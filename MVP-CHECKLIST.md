# MVP Checklist

## Objetivo

Lanzar en 7 días una V1 de `trabajo.iia.es` que:

- se pueda compartir con facilidad
- sea citable por periodistas y radios
- convierta visitante general en suscriptor de **1 Minuto de IA**

## Alcance V1

Entregables obligatorios:

- Home editorial en `trabajo.iia.es`
- Buscador principal
- Bloque "La noticia en 5 datos"
- Caja de suscripción integrada a **1 Minuto de IA**
- Sección "Para periodistas"
- Metodología clara y visible
- 6 fichas prioritarias
- Botones de compartir
- OG básicos

No entra en V1:

- Corte territorial
- Comparador de profesiones
- Analítica avanzada
- 62 fichas completas
- Automatizaciones complejas
- Widgets para medios

## Responsables sugeridos

- **Web / producto:** implementación estática, GitHub Pages, `CNAME`, diseño y maquetación
- **Editorial / datos:** claims, rankings, metodología, copy de fichas y revisión de rigor
- **Marketing 1:** outreach a economía, negocio, generalistas y radio
- **Marketing 2:** outreach a tecnología, empleo, RRHH, educación y creadores on-topic

## Reglas

- No inventar nada.
- Todo lo derivado por IA, heurística o estimación se etiqueta como tal.
- Si una pieza no llega con calidad, se recorta.
- Si hay duda entre feature y claridad editorial, gana claridad editorial.

## Día 1: Cerrar arquitectura

- [ ] Crear `CNAME` para `trabajo.iia.es`
- [ ] Confirmar despliegue estático en GitHub Pages
- [ ] Cerrar estructura final de la home
- [ ] Cerrar las 6 fichas V1
- [ ] Cerrar lista de claims permitidos
- [ ] Cerrar formato visible de metodología

Definition of done:

- `trabajo.iia.es` resuelve correctamente
- la estructura de página ya no cambia
- las 6 fichas quedan congeladas para V1

## Día 2: Home y conversión

- [ ] Montar hero editorial
- [ ] Añadir bloque "La noticia en 5 datos"
- [ ] Integrar caja principal de suscripción a **1 Minuto de IA**
- [ ] Integrar caja secundaria tras búsqueda o resultado
- [ ] Añadir branding IIA + referencia a `Explicable`

Definition of done:

- un visitante general entiende en 5 segundos qué es la página
- ve una CTA clara a suscribirse

## Día 3: Compartición y SEO básico

- [ ] Añadir botones WhatsApp, LinkedIn, X y copiar enlace
- [ ] Dejar URLs finales bajo `trabajo.iia.es`
- [ ] Corregir `title`
- [ ] Corregir `meta description`
- [ ] Añadir OG básicos
- [ ] Corregir BibTeX y placeholders

Definition of done:

- cualquier ficha o landing principal se puede compartir con una URL limpia
- la página deja de parecer un experimento suelto

## Día 4: Metodología y prensa

- [ ] Publicar metodología clara
- [ ] Explicar "exposición no significa sustitución"
- [ ] Crear sección "Para periodistas"
- [ ] Añadir email/contacto de prensa
- [ ] Preparar 3 titulares sugeridos
- [ ] Exportar 1 CSV descargable

Definition of done:

- un periodista puede entender la fuente, citarla y contactar

## Día 5: Fichas prioritarias

- [ ] Publicar ficha de Programador
- [ ] Publicar ficha de Administrativo
- [ ] Publicar ficha de Periodista
- [ ] Publicar ficha de Abogado
- [ ] Publicar ficha de Profesor
- [ ] Publicar ficha de Médico

Cada ficha debe incluir:

- [ ] titular SEO
- [ ] resumen en 3 líneas
- [ ] tareas automatizables
- [ ] tareas humanas
- [ ] qué aprender
- [ ] CTA a **1 Minuto de IA**

Definition of done:

- las 6 fichas son útiles, compartibles y no parecen páginas vacías

## Día 6: Outreach y activos

- [ ] Preparar nota de prensa corta
- [ ] Exportar 3 gráficos PNG
- [ ] Crear lista corta de medios objetivo
- [ ] Crear lista corta de creadores/newsletters objetivo
- [ ] Preparar 2 plantillas de pitch
- [ ] Revisar todos los envíos con supervisión humana

Targets sugeridos:

- economía / negocio
- tecnología
- empleo / RRHH
- radio nacional y radio económica
- newsletters y creadores muy on-topic

Definition of done:

- marketing puede empezar outreach sin improvisar

## Día 7: Lanzamiento y ajuste

- [ ] Publicar `trabajo.iia.es`
- [ ] Ejecutar outreach manual asistido
- [ ] Publicar en LinkedIn
- [ ] Publicar en X
- [ ] Revisar copy y CTA en móvil
- [ ] Corregir fallos visibles post-lanzamiento

Definition of done:

- la web está pública
- la campaña está en marcha
- no quedan errores de credibilidad obvios

## Criterio de recorte

Si falta tiempo, cortar en este orden:

1. gráficos extra
2. una o dos fichas menos
3. embellishments visuales
4. activos secundarios de campaña

No cortar:

- caja de suscripción
- metodología
- bloque editorial principal
- sección para periodistas
- dominio final `trabajo.iia.es`

- [ ] La lista de outreach está cerrada
